# Fundval API 文档

## 概述

Fundval 基金估值与资产管理平台 RESTful API。

## 基础信息

- **Base URL**: `http://localhost:8000/api`
- **协议**: HTTP/HTTPS
- **数据格式**: JSON
- **字符编码**: UTF-8

## HTTP 状态码

- `200 OK` - 请求成功
- `201 Created` - 创建成功
- `400 Bad Request` - 请求参数错误
- `401 Unauthorized` - 未认证
- `403 Forbidden` - 无权限
- `404 Not Found` - 资源不存在
- `500 Internal Server Error` - 服务器错误

## API 模块

1. [系统管理](./01-系统管理.md) - 健康检查、系统初始化
2. [用户认证](./02-用户认证.md) - 登录、注册、JWT 管理
3. [基金管理](./03-基金管理.md) - 基金列表、搜索、估值、准确率
4. [账户管理](./04-账户管理.md) - 账户 CRUD、账户持仓
5. [持仓管理](./05-持仓管理.md) - 持仓查询、操作流水、重算
6. [自选列表](./06-自选列表.md) - 自选列表 CRUD、添加/移除基金、排序
7. [数据源](./07-数据源.md) - 数据源列表、准确率统计
